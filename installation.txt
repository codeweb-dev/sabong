Step 1: download the file in github and extract

Step 2: open laravel herd and select the file

Step 3: open the file in vs code and then "composer install" & "npm install"

Step 4: next create the .env file do this in command "cp .env.example .env"

Step 5 Part 1:
    - then edit the .env the database, reverb, remember the sync etc.
        BROADCAST_CONNECTION=reverb
        BROADCAST_DRIVER=reverb
        QUEUE_CONNECTION=sync

Step 5 Part 2:
    - for credential paste this in .env
        REVERB_APP_ID=my-app-id
        REVERB_APP_KEY=my-app-key
        REVERB_APP_SECRET=my-app-secret
    - then run in command php artisan reverb:install
    - once you see the credential ctrl+c to stop the install

step 6: (do this one time now no need to do in next project update) php artisan migrate:fresh --seed

Step 7: open command again and type "php artisan key:generate"

Step 8: open again command and do "npm run build"

Step 9: is change the url and for example default
        - APP_URL=http://localhost to APP_URL=http://your-ip-address
        - REVERB_HOST=your-ip-address
        - REVERB_PORT=8080
        - VITE_REVERB_HOST="your-ip-address"

Step 10: run the system
    - php artisan serve --host=0.0.0.0 --port=8000 (Terminal 1)
    - php artisan reverb:start (Terminal 2)
